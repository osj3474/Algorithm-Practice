7. stack, queue

   [python]

   ```python
   from collections import deque
   
   # ====== stack ====== #
   s = []       
   s.append(1)
   s.pop()
   
   # ====== deque ====== #
   q = deque()
   q.append(1)
   q.popleft()
   ```

   [java]

   ```java
   import java.util.*
   
   // ====== stack ====== //
   Stack<Integer> s = new Stack<>();
   s.push(1);
   s.pop();
   s.peek();
   s.size();
   s.empty();    // 비우기
   s.isEmpty();  // 비었는지 확인
   
     
     
   // ====== deque ====== //
   Queue<Integer> q = new LinkedList<Integer>();
   q.add(1);
   q.remove();
   q.isEmpty();
   
   ArrayDeque<Integer> q = new ArrayDeque<Integer>();
   q.add(1);
   q.poll();
   q.addFirst(next);
   q.addLast(next);
   q.isEmpty();
   ```
